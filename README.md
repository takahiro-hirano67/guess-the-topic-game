# お題当てゲーム (Guess the Topic Game)

LLM（大規模言語モデル）がシステムプロンプトで保持している「お題」を、ユーザーが質問を通じて当てるWebアプリケーションです。

## 概要

このゲームでは、AIがゲームマスターとなり、心の中で決めたある「お題」を隠し持っています。ユーザーはAIに質問を投げかけ、その回答からお題を推測します。AIはルールに従い、お題を直接明かさずに、質問に対して最小限の回答を行います。

## 遊び方

1. アプリケーションを起動し、チャット画面を開きます。
2. 「それは食べ物ですか？」「生き物ですか？」など、お題を絞り込むための質問をAIに送ります。
3. ヒントを集め、確信が持てたら「私の予想は〇〇です」と伝えてください。
4. 正解するとAIがお題を公開します。
5. 途中で諦めたくなった場合は「答えを見る」ボタンでお題を確認できます。
6. 「リセット」ボタンを押すと、新しいお題でゲームをやり直せます。

## セットアップと起動方法

### 1. 環境変数の設定
`.env` ファイルを作成し、LLMの接続情報を設定してください。
```env
LLM_BASE_URL=あなたのLLMベースURL
LLM_API_KEY=あなたのAPIキー
LLM_NAME=使用するモデル名
FLASK_SECRET_KEY=任意のセッション用秘密鍵
```

### 2. 起動
以下のコマンドでサーバーを起動します。
```bash
python app.py
```
起動後、ブラウザで `http://127.0.0.1:5000` にアクセスしてください。

## プロジェクト構成

- `app.py`: Flask Webサーバーのメイン処理（ルーティング、セッション管理）
- `game.py`: ゲームのコアロジック（LLMとの対話制御）
- `config.py`: 環境変数およびLLMクライアントの設定
- `system_prompt.py`: AIゲームマスターへの詳細な指示（システムプロンプト）
- `topics.py`: お題のリストとランダム抽出ロジック
- `templates/index.html`: チャットUIのHTMLテンプレート
- `static/style.css`: チャットUIのスタイル定義
- `pyproject.toml`: プロジェクトのメタデータと依存関係定義
